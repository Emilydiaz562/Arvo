ARVO — Portafolio Creativo (Minimalista blanco + verde ARVO)

Archivos:
- index.html
- styles.css
- script.js
- /assets/Portafolio_ARVO.pdf  (tu PDF subido)
- /fotos/   (coloca aquí tus imágenes: foto1.jpg, foto2.jpg...)
- /videos/  (coloca aquí tus mp4: video1.mp4...)

Cómo probar localmente:
1. Abre index.html en un navegador o usa Live Server en VS Code.
2. Reemplaza las imágenes y videos en /fotos y /videos.
3. Para cambiar el número de WhatsApp, edita el enlace en index.html (busca wa.me).

Despliegue rápido (recomendado):
- GitHub Pages:
  1. Crea un repo en GitHub y sube todo el contenido (o sube el ZIP en la sección "Upload files").
  2. En Settings -> Pages -> Source selecciona branch 'main' y carpeta '/ (root)'. GitHub publicará tu sitio en https://tu-usuario.github.io/tu-repo/

- Netlify:
  1. Crea cuenta en netlify.com, arrastra el contenido de esta carpeta y suéltalo en 'Sites -> New site -> deploy manually'.
  2. Netlify generará una URL pública y HTTPS automáticamente.

Si quieres, puedo desplegarlo por ti conectando tu GitHub o indicándote paso a paso.
